About the Game

Piazza Panic is a cooking game made by Team Hard Cubans (THC) developed using the LibGDX Game Engine.

Controls

You can select between 3 different chefs using the "1", "2", and "3" keys to cook different dishes requested by customers.

Movement is done using the WASD and Arrow keys.

You have 4 different stations where the cooking is done.

To interact with the stations by pressing the "E" key

Note: You can only assemble the food item that the current customer wants
